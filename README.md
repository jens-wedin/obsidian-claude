# Obsidian Claude Assistant

AI-powered writing assistant plugin for [Obsidian](https://obsidian.md/) using Claude. Select text to improve, translate, summarize, and more.

## Features

- Text improvement and rewriting
- Translation
- Summarization
- Custom prompts

## Development

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm

### Setup

```bash
npm install
```

### Build

```bash
npm run build
```

### Development mode (watch for changes)

```bash
npm run dev
```

### Installing in Obsidian

1. Build the plugin with `npm run build`
2. Copy `main.js`, `manifest.json` to your vault's `.obsidian/plugins/obsidian-claude/` directory
3. Enable the plugin in Obsidian settings
